I also built a Run-Off Triangle tool to automate traditional reserving techniques. The tool calculates development factors, and projects future claims by completing the triangle- all within a streamlined, user-friendly structure using HTML and CSS
This project was inspired by the vacation work I completed in November 2024. I was curious to see whether actuarial reserving methods could be recreated using web technologies, and it became an exciting way to combine my technical interests with actuarial thinking.
